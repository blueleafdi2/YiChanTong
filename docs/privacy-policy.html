隐私政策

Privacy Policy · 遗产通 (YiChanTong) · 开发者：王迪

欢迎使用遗产通（以下简称"本应用"）。本隐私政策由开发者王迪（以下简称"我们"）制定，旨在向您说明我们如何收集、使用、存储和保护您的个人信息。请您仔细阅读本政策。

1. Data Collection / 信息收集

本应用采用最小化数据收集原则。我们收集或存储的信息包括：

  • 本地存储数据：收藏、笔记、阅读历史等数据仅存储在您设备本地（通过 SharedPreferences 实现），不会上传至任何服务器。
  • AI 对话数据：当您使用 AI 法律助手功能时，您输入的咨询内容会发送至 DeepSeek AI API 以获取回复。该数据仅用于处理您的即时请求，我们不会长期存储或用于其他目的。
  • 不收集：我们不会收集您的身份信息、位置信息、通讯录或设备标识符。

2. Information Usage / 信息使用

我们收集的信息仅用于以下目的：

  • 提供应用核心功能（收藏、笔记、阅读历史等本地功能）；
  • 在您使用 AI 法律助手时，将您的问题发送至 DeepSeek API 以生成回复；
  • 改进应用体验和功能。

3. Third-Party Services / 第三方服务

本应用使用以下第三方服务：

  • DeepSeek AI API：用于 AI 法律助手功能。当您使用该功能时，您输入的内容会发送至 DeepSeek 的服务器进行处理。请参阅 DeepSeek 的隐私政策以了解其数据处理方式。

除上述服务外，本应用不集成其他第三方数据分析、广告或追踪服务。

4. Data Security / 数据安全

  • 本地存储：收藏、笔记、阅读历史等数据仅保存在您设备本地，不进行云端同步。
  • 无账号系统：本应用不要求注册账号，因此不会存储您的账号或密码信息。
  • 数据删除：卸载应用后，本地存储的数据将被清除。

5. Children's Privacy / 儿童隐私

本应用不面向 14 周岁以下儿童。我们不会故意收集儿童的个人信息。如您发现我们无意中收集了儿童信息，请立即联系我们，我们将尽快删除。

8. User Rights / 用户权利

您享有以下权利：

  • 知情权：了解我们如何收集和使用您的信息；
  • 删除权：通过卸载应用清除本地存储的所有数据；
  • 选择权：您可以选择不使用 AI 法律助手功能，以避免向第三方发送数据。
