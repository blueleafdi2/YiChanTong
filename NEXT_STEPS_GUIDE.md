## 第一阶段：公司基础设施（第1-2周）

> 所有后续操作的前提。对公账户是迁移小程序的硬性要求。

### 1.1 刻章（如未完成）

深圳2025年起新设企业可**免费**刻制4枚印章（行政章、财务章、发票章、法人章）。

**操作路径**：
1. 登录 [深圳市开办企业一窗通](https://amr.sz.gov.cn/) → 选择"刻章服务"
2. 选择距离最近的刻章点，领取芯片印章
3. 系统自动完成公安备案

> 来源：深圳市市场监督管理局《开办企业一窗通》FAQ

### 1.2 银行开户（对公账户）⭐ 最优先

**为什么必须先做**：小程序迁移企业、企业认证、Apple开发者注册等全部需要对公账户。

**推荐银行**：招商银行（深圳本地服务好、线上化程度高、年费相对低）或中国银行（国有行稳妥）。

**所需材料**：
| 材料 | 说明 |
|------|------|
| 营业执照正副本原件 | 电子版不行，必须原件 |
| 法人身份证原件 | 王迪本人身份证 |
| 公章 + 财务章 + 法人私章 | 3枚印章全带 |
| 公司章程 | 工商注册时留存的那份 |
| 办公地址证明 | 租赁合同 或 近2个月水电费单（部分银行要求） |

**流程**：
1. **预约**：通过"开办企业一窗通"平台预约银行网点，或直接到银行柜台
2. **法人亲自到场**（不可代办），填写《开户申请书》等表格
3. 银行提交人民银行审核，约 **3-15个工作日** 出结果
4. 审核通过后领取"开户许可证"和网银U盾
5. **激活**：按银行提示完成首笔入账（通常1元即可）

**年费**：基本户 800-1500元/年（各行不同，可问有无减免）

> 来源：深圳市市场监督管理局；各商业银行官网

### 1.3 税务激活 + 记账报税

**重要：公司成立后30天内必须完成税务开业，即使没有收入也要每月/每季"零申报"！不报税会被罚款。**

**操作步骤**：
1. 登录 [深圳市电子税务局](https://etax.shenzhen.chinatax.gov.cn/)
2. 法人实名登录 → 【我要办税】→【综合信息报告】→【新办纳税人开业】
3. 完成后系统自动进行税费种认定
4. 办理「新办纳税人套餐」：会计制度备案 + 发票票种核定

**记账报税方案（二选一）**：

| 方案 | 费用 | 适合 |
|------|------|------|
| 代理记账公司 | 200-500元/月 | 不想操心，省心省力 |
| 自行申报 | 0元 | 有基础财税知识，前期零申报简单 |

**零申报操作**：每季度（1/4/7/10月）15日前登录电子税务局 → 增值税申报 → 全填0 → 提交；企业所得税同理。

**必做日历**：
- **每季度15日前**：增值税+企业所得税申报（零申报也要提交）
- **每年6月30日前**：工商年报（国家企业信用信息公示系统）
- **每年5月31日前**：企业所得税汇算清缴

> 来源：国家税务总局深圳市税务局；深圳注册公司指引

### 1.4 社保 & 公积金开户（暂可延后）

如果暂时只有法人自己、没有雇佣员工，可以暂不开通。等有招聘需求时再办。

### 1.5 域名注册 + ICP备案 + 企业官网

**为什么需要**：Apple Developer要求企业有可访问的官网；隐私政策需要托管在企业域名上；应用商店审核需要。

**推荐方案（最省钱）**：

| 项目 | 推荐 | 费用 |
|------|------|------|
| 域名 | 腾讯云 `.com` | 首年0元（新客）/续费约70元/年 |
| 域名备案 | 腾讯云免费代备案 | 0元（约5-15工作日） |
| 托管/主机 | 腾讯云轻量应用服务器 | 38元/年起（最低配） |
| SSL证书 | 腾讯云免费DV证书 | 0元 |

**建议域名**：`diyingtech.com` 或 `yichangtong.com` 或 `yichangtong.cn`

**操作步骤**：
1. 注册腾讯云账号 → [域名注册](https://cloud.tencent.com/product/dm)
2. 搜索并购买域名
3. 发起 [网站ICP备案](https://cloud.tencent.com/product/ba)（需营业执照、法人身份证、域名证书）
4. 备案通过后，部署一个简单的企业官网页面（含公司简介+隐私政策+联系方式）
5. 配置HTTPS（申请免费SSL证书）

**企业官网最简方案**：用一个静态HTML页面部署到轻量服务器，内容包括：
- 公司简介（深圳市迪莹互联网技术有限公司）
- 产品介绍（遗产通）
- 隐私政策（完整版）
- 联系方式（邮箱）
- 页脚放ICP备案号

> 来源：腾讯云官网；工信部ICP备案指引

## 第二阶段：小程序企业化迁移（第2-3周）

> 前提：对公账户已开通

### 2.1 方案选择：迁移 vs 注销重注册

| 对比项 | 迁移（推荐✅） | 注销+重新注册 |
|--------|---------------|---------------|
| 已有备案 | 变更备案即可 | 需重新备案（15-20工作日） |
| 已发布代码 | 保留 | 全部丢失 |
| 用户数据 | 保留 | 全部丢失 |
| AppID | 保持不变 | 新的AppID |
| 费用 | 300元迁移费 | 0元注册费 |
| 耗时 | 1-3工作日 | 3-4周 |

**强烈建议选择迁移**，保留现有一切。

### 2.2 迁移前准备

- [ ] 对公账户已开通且可接收打款
- [ ] 营业执照扫描件（彩色清晰）
- [ ] 法人（王迪）身份证正反面照片
- [ ] 确认小程序当前状态正常（未被封禁）
- [ ] 如有微信支付商户号 → 先解绑（你目前应该没有，跳过）

### 2.3 发起迁移

**操作路径**：登录 [微信公众平台](https://mp.weixin.qq.com/) → 设置 → 基本设置 → 主体信息 → 迁移小程序

1. **原主体验证**：管理员（你）扫码确认，输入注册手机号收到的验证码
2. **填写目标企业信息**：
   - 企业名称：深圳市迪莹互联网技术有限公司
   - 统一社会信用代码：（你营业执照上的18位编码）
   - 法定代表人：王迪
   - 对公账户信息：填写开户行+账号
3. **对公打款验证**：微信会向你的对公账户打入 0.01-0.99 元随机金额，你需要登录网银查看金额并在页面填入确认
4. **支付迁移费**：300元（微信支付）
5. **等待审核**：1-3个工作日
6. **迁移完成**：新管理员（王迪的微信）扫码激活

> 来源：微信开放文档《小程序主体迁移》；微信开放社区帖子

### 2.4 完成企业认证（迁移后30天内）

**操作路径**：微信公众平台 → 设置 → 微信认证 → 开始认证

1. 选择认证主体类型：**企业**
2. 上传材料：
   - 营业执照扫描件
   - 法人身份证正反面
   - 对公账户信息（用于验证打款）
   - 管理员手持身份证照片（部分情况）
3. 签署《微信认证服务协议》
4. 选择发票类型（建议选电子发票）
5. 支付认证费：**300元/年**
6. 等待审核：约 **7个工作日**
7. 审核期间可能接到 0755 开头的电话核实，**务必接听**

**认证通过后获得的能力**：
- ✅ **被搜索** — 微信搜一搜可搜到你的小程序
- ✅ **被分享** — 分享卡片展示认证标识
- ✅ 开通微信支付
- ✅ 获取用户手机号等高级接口
- ✅ 名称保护（他人不能用相同名称）
- ✅ 企业级客服消息能力

> 来源：微信开放文档《微信认证指引》

### 2.5 改名为"遗产通" + 变更备案

**改名操作**（认证通过后）：

1. 微信公众平台 → 设置 → 基本设置 → 小程序名称 → **修改**
2. 管理员扫码验证
3. 输入新名称：**遗产通**
4. 等待名称审核（通常1-2工作日）

**为什么企业主体改名"遗产通"大概率能通过**：
- 你的经营范围包含「信息技术咨询服务」「软件开发」「社会经济咨询服务」
- 企业主体不受"个人备案名称限制"
- "遗产通"与你的实际服务内容强关联
- 如果名称被判定为通用词，可发邮件至 miniprogram@tencent.com 申诉，附上营业执照

**变更备案**（改名后必须做）：

1. 微信公众平台 → 设置 → 小程序备案 → **变更**
2. 更新小程序名称、主体信息（改为企业信息）
3. 提交审核 → 平台初审1-2工作日 → 管局审核1-20工作日
4. 收到工信部12381短信后，24小时内到 [beian.miit.gov.cn](https://beian.miit.gov.cn/) 完成短信核验

**备注建议文案（企业版，可以写得充实）**：

```
遗产通是深圳市迪莹互联网技术有限公司开发的法律知识查询工具，主要提供中国遗产继承相关法律法规的学习与检索服务。功能包括：民法典继承编法条查阅、全国公开裁判案例浏览、法律术语词典、继承关系计算器、遗产清单模板等实用工具，以及AI智能问答助手。所有内容来源于公开法律法规和中国裁判文书网公开判例，不提供律师委托或法律代理服务。
```

> 来源：微信开放文档《备案常见问答》第九/十五节；微信开放社区经验帖

## 第三阶段：功能恢复与升级提审（第3-4周）

> 前提：企业认证已通过，名称已改为"遗产通"

### 3.1 恢复被删减的功能

企业主体 + 已认证后，之前为了过审删减的功能可以恢复：

#### ✅ 可以直接恢复的（大概率无风险）：

| 功能 | 说明 | 过审风险 |
|------|------|---------|
| **个人微信联系** | 恢复QR码+微信号复制，但改标题为「联系开发者」而非「咨询法律专家」 | 低 — 企业提供客服联系方式是正常的 |
| **AI智能助手** | 恢复首页/工具箱/我的3个入口 | **需先申请"深度合成"类目**（见3.2） |
| **备注/备案中提及AI** | 企业主体可以写 | 低 — 经营范围含"人工智能应用软件开发" |

#### ⚠️ 恢复时需注意措辞的：

| 原措辞 | 建议改为 | 原因 |
|--------|---------|------|
| 咨询法律专家 | 联系开发者 | 避免暗示提供法律服务 |
| AI法律助手 | AI智能问答 | 淡化"法律服务"色彩 |
| 法律咨询 | 法律知识查询 | 强调工具属性 |

### 3.2 AI功能上线的前置条件（深度合成类目）

这是恢复AI功能的**关键步骤**。你的经营范围包含「人工智能应用软件开发」，这非常有利。

**申请流程**：

1. **获取DeepSeek的算法备案证明**：
   - DeepSeek（深度求索）已完成国家算法备案
   - 算法备案编号：网信算备310104944869801240019号
   - 需要获取DeepSeek的《在用证明》或《合作协议》
   - 联系方式：到 [DeepSeek开放平台](https://platform.deepseek.com/) 查看合作/资质文档，或发邮件咨询

2. **在微信后台申请类目**：
   - 微信公众平台 → 设置 → 基本设置 → 服务类目 → 添加
   - 选择：**深度合成 → AI问答**
   - 上传材料：
     - 营业执照
     - DeepSeek的算法备案截图或在用证明
     - 你与DeepSeek的合作关系说明（可以是API购买凭证+简要说明）

3. **审核通过后**恢复AI入口

**如果获取DeepSeek证明比较麻烦**，替代方案：
- 先不申请深度合成类目，AI功能暂时保持隐藏
- 其他所有功能先上线
- 等拿到证明后再单独发一个版本恢复AI

> 来源：微信开放文档《深度合成类目资质要求》；微信开放社区帖子

## 第四阶段：全平台上架（第4-8周）

### 4.1 Apple App Store（iOS）

**前置条件**：
- [x] 企业营业执照
- [ ] 对公账户
- [ ] 企业域名+官网
- [ ] D-U-N-S编码

**Step 1：申请邓白氏编码（D-U-N-S Number）— 免费，5个工作日**

1. 访问 [Apple D-U-N-S查询页面](https://developer.apple.com/enroll/duns-lookup/)
2. 搜索「Diying Internet Technology Co., Ltd.」（你营业执照上的英文名）
3. 如果没有 → 点击申请 → 填写公司英文名称、地址、法人英文名等
4. 等待 D&B 邮件确认（约5个工作日）
5. 拿到 D-U-N-S 编码后，等2个工作日同步到Apple系统

**Step 2：注册 Apple Developer — 688元/年**

1. 在 iPhone/iPad/Mac 上下载「Apple Developer」App
2. 用企业专属Apple ID登录（建议用 admin@你的域名.com）
3. 选择「组织/企业」→ 填入 D-U-N-S 编码
4. 法人人脸验证 + 身份证
5. 填写企业信息（名称必须与D-U-N-S英文名一致）
6. 支付 688元（支持支付宝/微信）
7. 等待Apple审核（3-5个工作日），期间可能接到Apple电话核实

**Step 3：上架 iOS App**

需要：
- 使用 Flutter/React Native 将现有安卓 App 移植为 iOS 版，或开发新的 iOS 版本
- 或考虑先不上 iOS（投入产出比较低），专注小程序+安卓

> 来源：Apple Developer官方文档；博客园注册攻略
