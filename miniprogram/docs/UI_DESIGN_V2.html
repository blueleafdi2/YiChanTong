遗嘱怎么写才有效？
根据《民法典》继承编，有效遗嘱需要满足以下条件：

1. 自书遗嘱（第1134条）：由遗嘱人亲笔书写，签名，注明年月日。
2. 代书遗嘱（第1135条）：应有两个以上见证人在场见证。
3. 打印遗嘱（第1136条）：应有两个以上见证人在场见证，遗嘱人和见证人应在每一页签名、注明日期。

⚠️ 无民事行为能力人或限制民事行为能力人所立的遗嘱无效。
